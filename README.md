# cart_spheroids_flow

## Descripción
Proyecto de análisis ómico en R.

## Ambiente
Activar con: `conda activate omics-R`

## Estructura
- data/raw: datos originales (no modificar)
- data/intermediate: archivos intermedios
- data/processed: datos finales
- scripts: scripts R (.R)
- notebooks: análisis interactivo (RMarkdown)
- results/figures: figuras generadas
- results/tables: tablas de resultados
- config: archivos de configuración
- docs: documentación
- logs: salidas de ejecución
- env: entorno reproducible

## Ejecución
Ver `docs/RUNBOOK.md` para pasos de reproducción.
